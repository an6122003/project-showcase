# Project Files

Each subfolder corresponds to a project in the portfolio.
Add your PDF files, images, and other assets to the relevant folder.

## Structure

```
projects/
├── zalopay-case-study/     # ZaloPay FinTech case study
│   ├── slide-1.pdf         # Case study PDFs
│   ├── slide-2.pdf
│   └── cover.png           # Optional cover image
├── retrolab/               # RetroLab AI Tech News
│   ├── architecture.pdf
│   └── cover.png
├── ecommerce-auction/      # E-Commerce Live Auction
│   ├── prd.pdf
│   └── cover.png
└── rbac-system-design/     # RBAC System Design
    ├── documentation.pdf
    └── cover.png
```

## Supported File Types
- **PDFs**: Case study documents, PRDs, slide decks
- **Images**: Cover images, screenshots, diagrams (PNG, JPG, WebP)
